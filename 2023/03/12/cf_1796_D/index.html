Educational Codeforces Round 144 (Rated for Div. 2) D. Maximum Subarray (dp)

dp

链接

https://codeforces.com/contest/1796/problem/D

题意

给一个数组 \(a\)，要求选出 \(a\) 的一个长度为 \(k\) 的子序列 \(b_{1}, b_{2}, ..., b_{k}\)，让 \(\forall i \in [1, k], b_{i} += x\)，而让 \(\forall a_{i} \in a \ and \ a_{i} \notin b, a_{i} -= x\)，使得形成的新数组最大子段和最大。

题解

考虑 DP。
设 \(f[i][j]\) : 考虑前 \(i\) 个数，已经选择了 \(j\) 个数执行 "\(+x\)" 操作的大子段和。
分类讨论：

  • \(a[i]\) 不执行 "\(+x\)" 操作
    • 必须满足 \(i > j\)，因为如果 \(i=j\) 则意味着前 \(i\) 个数全部执行了 "\(+x\)" 操作，因此 \(a[i]\) 必须执行 "\(+x\)" 操作
    • \(a[i]\) 必须执行 "\(-x\)" 操作
    • \(a[i]\) 单独作为一个子段和，即 \(a[i] - x\)
    • \(a[i]\) 合并到 \([1, i-1]\) 的最大子段和中，即 \(f[i-1, j] + a[i] - x\)
  • \(a[i]\) 执行 "\(+x\)" 操作
    • \(a[i]\) 单独作为一个子段和，即 \(a[i] + x\)
    • \(a[i]\) 合并到 \([1, i-1]\) 的最大子段和中，即 \(f[i-1, j-1] + a[i] + x\)

因此，状态转移方程为： \[ f[i,j]=\left\{\begin{matrix}max(a[i]-x,f[i-1,j]+a[i]-x)\ \ j<i \\max(a[i]+x, f[i-1, j-1]+a[i]+x)\ \ j > 0 \end{matrix}\right. \] 初始条件为：\(f[0, 0] = 0\)

python 代码

T = int(input())
for _ in range(T):
    n, k, x = [int(_) for _ in input().split()]
    a = [int(_) for _ in input().split()]
    inf = float('inf')
    f = [[-inf for _ in range(k+1)] for _ in range(n+1)]
    res = 0
    f[0][0] = 0
    for i in range(1, n+1):
        # 当前已经使用的 "+x" 操作 j 最少为 k-(n-i)，即让 i 后面的数全部执行 "+x" 操作
        # 当前已经使用的 "+x" 操作 j 最多为 i，即让 i 前面的数全部执行 "+x" 操作
        for j in range(max(0, k-(n-i)), min(i, k)+1):
            if j < i:
                f[i][j] = max(a[i-1] - x, f[i-1][j] + a[i-1] - x)
            if j:
                # 注意此处 f[i][j] 可能已经被更新，因此要带上
                f[i][j] = max(f[i][j], max(a[i-1] + x, f[i-1][j-1] + a[i-1] + x))
            # 若只考虑当前的最大子段和，其他的 "+x" 操作发生在什么地方对当前结果没有影响
            res = max(res, f[i][j])
    print(res)
