yolov5配置

项目配置教程

1. 安装与环境配置

  • 安装 Pycharm
    下载地址 https://www.jetbrains.com/zh-cn/pycharm/download/#section=windows (下载最新版即可)

  • 安装 Anaconda
    下载地址 https://www.anaconda.com/products/distribution/start-coding-immediately (下载最新版即可)
    此后所有命令都在 Anaconda Prompt (Anaconda3) 中运行

  • 创建虚拟环境
    打开 Anaconda Prompt (Anaconda3)
    创建虚拟环境： 

    conda create -n 自定义的名称(例如 vexento) python==3.7.2
    检查当前的虚拟环境： 
    conda env list
    如果 python 版本安装错误，则在 base 环境中输入 
    conda install python==3.7.2 -n 要修改的环境名(例如 vexento)

  • 更改 pycharm 的解释器

    --> File 
--> Settings 
--> Project 
--> Python Interprerter
--> Add Interpreter 
--> Add Local Interpreter 
--> Virtualenv Enviroment 中的 Existing 
--> ... 
--> 找到上一步虚拟环境地址中的 python.exe 
--> OK

  • 安装 cuda 11.7
    下载地址：https://developer.nvidia.com/cuda-11-7-1-download-archive?target_os=Windows&target_arch=x86_64&target_version=10&target_type=exe_local

默认安装即可

  • 安装 cudNN
    下载地址：https://developer.nvidia.com/rdp/cudnn-archive (选择对应版本 for CUDA 11.x)
    解压，将解压后的所有文件复制到 cuda 11.7 安装文件夹 

    C:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v11.7

  • 安装 Pytorch
    打开 Anaconda Prompt (Anaconda3)
    激活虚拟环境： 

    activate 虚拟环境的名称(例如 vexento)
    下载 Pytorch： 
      pip3 install torch torchvision torchaudio --extra-index-url https://download.pytorch.org/whl/cu117  --default-timeout=10000
  ```  
下载 torchvision：
    python -m pip install -i http://pypi.tuna.tsinghua.edu.cn/simple --trusted-host pypi.tuna.tsinghua.edu.cn torchvision 
      打开 Pycharm, 等待 Pycharm 更新解释器  
  
  - 下载 yolov5  
  下载地址：https://github.com/ultralytics/yolov5  
  解压缩

  - 安装需要的包  
- 进入 yolov5-master 文件夹：
    ```c
    cd yolov5-master的路径
    下载 yolov5 所需要的包： 
    pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple  --default-timeout=10000
